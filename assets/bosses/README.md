# Dossier des images de boss

Ce dossier contient les images des boss pour les événements de combat.

## Structure

- **Mini Boss** : Images 512x512 ou 256x256
    - `slime_geant.png` - Slime Géant
    - `golem_pierre.png` - Golem de Pierre
    - `esprit_corrompu.png` - Esprit Corrompu
    - `araignee_geante.png` - Araignée Géante

- **Boss** : Images 512x512 ou 256x256
    - `dragon_chaos.png` - Dragon du Chaos
    - `liche_roi.png` - Roi Liche
    - `hydre_abyssale.png` - Hydre Abyssale
    - `titan_forge.png` - Titan Forgé

## Instructions

Placez vos images de boss dans ce dossier en utilisant les noms exacts listés ci-dessus.

Les images doivent être au format PNG pour la transparence.

Si une image n'existe pas, l'événement fonctionnera quand même sans thumbnail.

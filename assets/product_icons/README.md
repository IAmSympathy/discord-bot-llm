# Product Icons

Ce dossier contient les icônes pour les différents types de produits FreeStuff (DLC, Loot, Software, etc.).

## Icônes Disponibles

| Fichier         | Type       | Description                          |
|-----------------|------------|--------------------------------------|
| `dlc.png`       | DLC        | Extension/Add-on de jeu              |
| `loot.png`      | Loot       | Butin in-game (coffres, récompenses) |
| `software.png`  | Software   | Logiciels/Applications               |
| `art.png`       | Art        | Contenu artistique                   |
| `ost.png`       | OST        | Bandes sonores                       |
| `book.png`      | Book       | Livres/Ebooks                        |
| `storeitem.png` | Store Item | Articles de boutique                 |
| `other.png`     | Other      | Autres types de produits             |

## Format

- **Format** : PNG avec transparence
- **Taille recommandée** : 512x512 pixels
- **Utilisation** : Thumbnail d'embed Discord

## Utilisation dans le Code

Ces icônes sont utilisées automatiquement par le service FreeStuff :

- Pour les **jeux** : Le logo de la plateforme (Steam, Epic, etc.) est utilisé
- Pour les **autres types** : L'icône correspondante de ce dossier est utilisée

## Remplacement des Icônes

Pour remplacer une icône :

1. Créez ou téléchargez une nouvelle icône (PNG, 512x512 recommandé)
2. Remplacez le fichier correspondant dans ce dossier
3. Redéployez le bot : `.\deploy-to-oracle.ps1`

## Sources
